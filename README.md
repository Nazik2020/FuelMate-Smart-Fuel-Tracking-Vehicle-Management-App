# FuelMate - Smart Fuel Tracking & Vehicle Management App

A comprehensive mobile application for tracking fuel consumption, managing vehicles, and locating nearby fuel stations.

## Features

- **Fuel Log Tracking** - Record and analyze fuel consumption patterns
- **Vehicle Management** - Manage multiple vehicles with detailed information
- **Fuel Station Finder** - Locate nearby fuel stations on an interactive map
- **Task Reminders** - Set maintenance and fuel-related reminders
- **User Dashboard** - View fuel statistics and spending analytics
- **Admin Panel** - Manage users, vehicles, and fuel prices (admin only)

## Technologies Used

- React Native with Expo Framework
- Firebase (Authentication, Firestore, Storage)
- React Navigation
- React Native Maps
- TypeScript

## Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npx expo start`
4. Scan QR code with Expo Go app or run on emulator

## Project Structure

- `/app` - Screen components and navigation
- `/components` - Reusable UI components
- `/config` - Firebase and service configurations
- `/hooks` - Custom React hooks
- `/constants` - Theme and constant values
